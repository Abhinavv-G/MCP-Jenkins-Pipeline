# MCP-Integrated Jenkins Pipeline

A CI/CD pipeline that integrates **Cursor AI** with a **Jenkins MCP (Model Context Protocol) Server** to automate build, testing, containerization, and deployment workflows.

## Features

* AI-driven pipeline execution using Cursor AI
* Jenkins MCP Server integration
* Automated build and testing
* Docker-based containerization
* Kubernetes deployment using Minikube
* Dynamic image generation and validation during testing
